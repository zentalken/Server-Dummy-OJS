Ringkasan shortcut Capstone (Server OJS), jalankan di Powershell/Terminal:

- Masuk ke folder: cd "C:\Joseph\SEMESTER 6\Capstone\ojs-server" (Masuk ke folder dmn docker-compose.yml berada)
- Buka aplikasi Docker Desktop
- Menjalankan Container: docker-compose up -d
- Menghentikan Container: docker-compose down
- Memastikan Container hidup: docker ps

- Frontend: http://localhost:8080
- Backend: http://localhost:8080/index/id/admin
